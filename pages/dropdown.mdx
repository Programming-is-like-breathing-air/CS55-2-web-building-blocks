# Dropdown Component

**Contributed by Nuojin Zhang**

Dropdown allow users to choose from a list of options in a limited space. The list of options can change based on the context.

## Basic Component

### Design:
This component utilizes the DropdownMenu and DropdownMenuCheckboxItem components from the UI library to create a dropdown menu with multiple selectable checkboxes. 

### How to behave:
- **Checkbox Interaction**: Clicking on a checkbox item toggles its state.
The toggleOption function manages the state of selected options in the dropdown menu.

### How to use it:
- Installation
Ensure that you have the necessary dependencies installed in your project. If not, install them using npm by running the following command in your terminal:
```
npx shadcn-ui@latest add dropdown-menu
npx shadcn-ui@latest add button
```

- Usage: Copy the code for the `DropdownMenuCheckboxes` component and import it into your React application where you want to use it.
```
import { DropdownMenuCheckboxItemProps } from "@radix-ui/react-dropdown-menu"

import { Button } from "@/components/ui/button"
import {
  DropdownMenu,
  DropdownMenuCheckboxItem,
  DropdownMenuContent,
  DropdownMenuSeparator,
  DropdownMenuTrigger,
} from "@/components/ui/dropdown-menu"
```

- Rendering the Component:Place the `DropdownMenuCheckboxes` component within your JSX to render it on the desired page or component.
- Customization:
      - Customize the appearance and behavior of the dropdown menu and checkbox items as needed.
      - Adjust the button variant, dropdown menu width, and styling to match your application's design and requirements.
      - **Example**: Use this component to create a dropdown menu for selecting multiple options, such as preferences, settings, or categories. Customize the options and functionality to suit your specific use case, such as filtering, sorting, or managing user preferences.

### With header
#### Design:
This component represents a dropdown menu with a header indicating the user's signed-in status. It utilizes the DropdownMenu component from the UI library to create the dropdown functionality. The dropdown menu contains several menu items for different options, such as account settings, support, license, and sign-out.

#### How to behave:
- Menu Interaction:
Clicking on the button labeled "Option" triggers the dropdown menu to appear.
The dropdown menu displays a header labeled "Signed in as" followed by the user's email address.
- Below the header, several menu items are listed, including account settings, support, license, and sign-out.
- Clicking on any menu item performs the corresponding action, such as navigating to a settings page or logging out of the application.
