# Pagination Component

**Contributed by Yingjian Di**

This is a function that helps users select page numbers.

## Basic Component
### Design:
The `PaginationDemo` component is a React component tailored for managing page navigation within applications that require content to be divided across multiple pages. This component leverages styled components like `Pagination`, `PaginationContent`, `PaginationItem`, `PaginationLink`, `PaginationPrevious`, `PaginationNext`, and `PaginationEllipsis` from a UI library. These elements work together to offer a user-friendly pagination system that facilitates easy access to various pages of content.
### How to behave:
- Click: User can click the button to go to the next or previous page
- Selection: Users can directly select page numbers
### How to use it:
- Installation
Ensure that your project includes all necessary dependencies for the `pagination` component. If they are not already installed, add them via npm with the following command: 
```
npx shadcn-ui@latest add pagination 
```
- Usage:
To integrate the `pagination` into your React application, include the necessary imports from the Shadcn UI library:
```
import {
  Pagination,
  PaginationContent,
  PaginationEllipsis,
  PaginationItem,
  PaginationLink,
  PaginationNext,
  PaginationPrevious,
} from "@/components/ui/pagination"


```

```
<Pagination>
  <PaginationContent>
    <PaginationItem>
      <PaginationPrevious href="#" />
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="#">1</PaginationLink>
    </PaginationItem>
    <PaginationItem>
      <PaginationEllipsis />
    </PaginationItem>
    <PaginationItem>
      <PaginationNext href="#" />
    </PaginationItem>
  </PaginationContent>
</Pagination>


```
- Logic:
```
const handlePageChange = (event, page) => {
    event.preventDefault();
    setCurrentPage(page);
  };
```
- Customization:
Modify the total number of pages by setting `totalPages`. Manage the status of the current page number by setting `useState(1)`. Set the maximum number of page numbers displayed simultaneously by modifying `maxPageNumbersToShow`.
```
const totalPages = 10; // Total number of pages.
const [currentPage, setCurrentPage] = useState(1);
const maxPageNumbersToShow = 5; // Maximum number of page numbers to display at once.

```
